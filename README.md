# GunGame 1.6

2D шутер з виглядом з боку

## Управління в грі:
* ЛКМ - стріляти
* ←, →, ↑, ↓ - ходити

## Фичи:
 * 3 вида зброї
 * апгрейди за монетки
 * AI 

## Використані інструменти:
 * C#
 * Unity
 * 2d Animations
 * 2d physics
 * UI
